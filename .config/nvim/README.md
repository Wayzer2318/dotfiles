# moving in nvim
* To scroll up and down with half the length of the screen use "ctrl u" for up and "ctrl d" for down 
* if u dont use relative number lines u can go to a line by entering the number of the line and then "gg"
* i u use relative number lines u can say that u want to move 6 line up bye pressing 6 then 'j' for down and 'k' for up
* use 'zz' on the actual line to center in the middle of ur screen 
* use "shift W" and "shift B"  to go through text ( it will ignore special caracters)
* if u use 'w' u will do the same but word by word and not ignore special caracters
* use 's' to move to a specific caracter with flash.nvim / u can also use ';' and ',' to move forward and backward
* use '/' to search in the whole file with treesseter
* use an operation (v, d, y, c) and 'i' for inside or 'a' for around and then the parameter that u wanna operate (ex v i " to select inside "") / or  "shift R" to operate on a search selection
* when u move, its added to jump list, to move incide the jump list use "ctrl o" or "ctrl i"
 u can use ctrl space to go to visual mode and select what u want
